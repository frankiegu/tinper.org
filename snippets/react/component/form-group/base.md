# bee-form-group

`FormGroup` 组件用来包裹像 `form control` `lable` `help text` `validate state`的元素


## API

| 参数        | 说明                                       | 类型     | 默认值  |
| --------- | ---------------------------------------- | ------ | ---- |
|validationState|oneOf:`success` `error` `warning`|string|''|
