# 月份选择

本例展示:月份UI控件绑定默认数据。

[试一试](http://tinper.org/webide/#/demos/kero/month)


**注**：设置月份时，通过dataTable中的对应行row调用setValue设置字段值即可。eg：`row.setValue("f1", "11")`
