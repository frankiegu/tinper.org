# bee-input-group-addon

InputGroupAddon 作为FormControl的边标签容器存在

## 使用

```js
import InputGroupAddon from 'tinper-bee';
ReactDOM.render(
    <div>
        <InputGroupAddon/>
    </div>,
    document.getElementById('target'));
```
