NeoUI

NeoUI 是基于 UI 设计语言 iuap design 实现的前端框架，致力于快速构建前端页面。

全局CSS样式

CSS样式提供了基础样式、布局、色彩、辅助类、字体图标、以及基本控件。开发者只需调用相应的样式即可，操作很简便。

组件

把一些常见的网页组件拆分成不同的部分，进行类似 Web Components 的封装，包括分页、下拉菜单、进度条、弹出框等更多功能。

Javascript插件

20+个官方插件，并在持续开发中，满足你的功能需求。使用方式及其简单，具体细节请查看各个插件文档
