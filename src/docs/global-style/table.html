table控件

表格控件提供了丰富的表格样式，其中有常用表格（u-table）、基本表格（u-table-base）、悬浮表格、边框表格、斑马线表格

常用表格：在基本表格的基础上添加了边框、表头背景色、悬浮特效。

其余表格控件样式也都是基于基本表格来实现的。
