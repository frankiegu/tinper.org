## # id
元素对应的id

## # data
元素对应的dateTable

## # field
元素对应的dateTable中的字段

## # type
元素对应的类型，此处u-time表示元素为时间控件，显示为输入面板进行value设置。

type为u-clockpicker同样为时间控件，显示为钟表形式进行value设置。
