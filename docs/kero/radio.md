## # id
元素对应的id

## # data
元素对应的dateTable

## # field
元素对应的dateTable中的字段

## # type
元素对应的类型，此处u-radio表示元素为单选。若多个元素设置的data与field一致，则表示为单选组，选择过程中只能选择其中一项。
