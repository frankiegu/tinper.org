# 徽章控件

徽章控件主要包括不同色彩的徽章、含有图标的徽章。

# 如何使用

添加含有`u-badge`就可以创建一个徽章，徽章的内容在data-badge中设置。

# 示例


##绝对定位标记
用于推送消息或提醒时，在标签dom元素中添加添加具体的图标即可。

    display: inline-block;
}
</style></div>

</div>

    <i class="uf uf-bellmusicaltool"></i>
</div></div>
