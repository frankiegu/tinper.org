# tabs控件

多内容分类切换显示

# 插件依赖

依赖于 http://design.yyuap.com/static/uui/latest/js/u.js

# 用法

定义锚链接，与内容id匹配，详情见示例



# 示例


##背景tabs

切换 背景色跟随作为`active`状态
